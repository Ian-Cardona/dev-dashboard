# Change Log

All notable changes to the "vscode-extension" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [1.0.0] - 2025-12-14

### Added

- Automatic scanning and organization of `// TODO:` comments across open VS Code workspaces
- On-demand synchronization of TODOs to the Dev Dashboard web app
- API key configuration for secure user authentication and data syncing
- Seamless integration with the Dev Dashboard for centralized development task management
